# vue-vuex

## 用法

克隆仓库到本地

```
git clone https://github.com/heuuLZP/vue-vuex.git
```

进入仓库根目录,安装依赖
```
npm install
```

启动项目
```
npm run dev
```



## 项目结构

```
src
├── App.vue
├── assets
│   └── logo.png
├── components        ## 组件
│   ├── Home.vue      ## 父组件
│   ├── add.vue       ## 子组件
│   └── reduce.vue    ## 子组件
├── main.js
├── router            ## 路由
│   └── index.js
└── vuex              ## 状态管理
    └── store.js
```
